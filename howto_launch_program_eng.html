How-To

A how-to is an informal, often short, description of how to accomplish a specific task. A how-to is usually meant to help non-experts, may leave out details that are only important to experts, and may also be greatly simplified from an overall discussion of the topic.

One of the earliest how-to books was published in 1569 and entitled, A booke of the arte and maner, how to plant and graffe all sortes of trees: With divers other new practise, by one of the Abbey of Saint Vincent in Fraunce by Leonard Mascall.

Perhaps the best known full-length book in the genre is How to Win Friends and Influence People, written by Dale Carnegie in 1936.

A similar concept can be seen in many of the [topic] For Dummies series of tutorials and also in many other introductory surveys entitled with the suffix "101" (based on academic numberings of entry-level courses).

How To Launch PHP Program using JavaScript

Hello.

It is Possible to Launch PHP Program by Passing Parameters from Javascript Program using GET and POST Methods, Both are Described Below.

Both Methods use removeBom function, this function remove unnecessary BOM (Byte Order Mask) from the Beginning of the PHP Responce, this is Some Bug, i have no Idea Where it is in the Responce From....

Difference Between GET and POST methods is following - GET Method is Reasonable to Use if Length of Parameters, send by GET is relatively small (<1000 Characters), and POST is is Reasonable to Use if Length of Parameters is big (>1000 Characters).

1. Get Method, The JavaScript File: 

function launchProgram(pattern) {
	if (window.XMLHttpRequest) {
		// code for IE7+, Firefox, Chrome, Opera, Safari
		xhr=new XMLHttpRequest();
	} else {  // code for IE6, IE5
		xhr=new ActiveXObject("Microsoft.XMLHTTP");
	}
	xhr.onreadystatechange = function(){
		if (this.readyState==4 && this.status==200) {

			dir = JSON.parse(removeBom(this.responseText));
			// further processing here
		}
	};
	xhr.open("GET","scripts/php/dir.php?q="+"../../"+encodeURIComponent(pattern),true);
	xhr.send();
}


1. Post Method, The JavaScript File: 

function launchProgram(pattern) {
	if (window.XMLHttpRequest) {
		// code for IE7+, Firefox, Chrome, Opera, Safari
		xhr=new XMLHttpRequest();
	} else {  // code for IE6, IE5
		xhr=new ActiveXObject("Microsoft.XMLHTTP");
	}
	xhr.onreadystatechange = function(){
		if (this.readyState==4 && this.status==200) {

			dir = JSON.parse(removeBom(this.responseText));
			// further processing here
		}
	};

	xhr.open("POST", "scripts/php/dir.php", true);
	xhr.setRequestHeader("Content-type", "application/x-www-form-urlencoded");
	xhr.send("q=../../"+ encodeURIComponent(pattern));
}


The PHP file (dir.php): 

<?
//get the q parameter from URL
$pattern=$_GET["q"];

foreach (array_filter(glob($pattern), 'is_file')  as $filename) {
	// pathinfo:
	// ['basename'] - Name + ext
	// ['extension'] - ext
	// ['filename'] - Name
	// ['dirname'] - Directory with LEADING ../
	$out[] = pathinfo($filename);
}
echo (json_encode($out)); 
?>
